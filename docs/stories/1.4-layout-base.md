# Story 1.4 — Layout Base (Header, Footer, Estrutura de Rotas)

## Descricao
Como usuario, preciso ver um layout base limpo e minimalista inspirado no Color Hunt, com header fixo, navegacao por categorias e footer discreto, para ter uma experiencia visual consistente em todas as paginas.

## Criterios de Aceitacao
- [ ] Root layout com fonte Inter, metadata global e estrutura HTML
- [ ] Header fixo com logo "Color" a esquerda e login placeholder a direita
- [ ] Navegacao por categorias: Trending, Popular, New, Random
- [ ] Footer minimalista com copyright e links
- [ ] Pagina home com placeholder "grid de paletas aqui"
- [ ] Layout responsivo (mobile-first)
- [ ] Espacamento generoso entre elementos

- [ ] Estilizar navegacao: pills/tabs com hover sutil
- [ ] Testar responsividade em mobile e desktop

## Dependencias
- Story 1.2 (Tailwind + shadcn/ui configurados)

## Notas Tecnicas
- Header: position sticky, top 0, z-50, bg-white, border-bottom sutil
- Logo: texto simples "Color" em font-semibold, sem imagem
- Categorias: inline-flex gap-2, texto cinza, hover com underline ou bg sutil
- Footer: py-8, text-center, text-gray-400, text-sm
- Max-width do conteudo: max-w-6xl mx-auto px-4

## Referencia Visual
- Color Hunt: header fino, clean, logo texto, categorias como tabs
