# Story 1.1 — Setup Next.js 15 + Cloudflare Workers

## Descricao
Como desenvolvedor, preciso inicializar o projeto Next.js 15 com App Router configurado para rodar no Cloudflare Workers via @opennextjs/cloudflare, para que toda a aplicacao rode na borda com latencia zero.

## Criterios de Aceitacao
- [ ] Next.js 15 inicializado com App Router (TypeScript)
- [ ] @opennextjs/cloudflare instalado e configurado
- [ ] wrangler.toml configurado com bindings (D1, KV, R2, Cron Triggers)
- [ ] `.dev.vars` criado com variaveis de ambiente placeholder
- [ ] `npm run dev` funciona localmente
- [ ] `npx opennextjs-cloudflare build` gera build sem erros
- [ ] .gitignore configurado corretamente

## Tarefas
- [ ] `npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir`
- [ ] `npm install @opennextjs/cloudflare wrangler`
- [ ] Criar `wrangler.toml` com bindings D1, KV, R2 e Cron Triggers
- [ ] Criar `open-next.config.ts`
- [ ] Criar `.dev.vars` com placeholders
- [ ] Adicionar scripts no package.json: `build:cf`, `deploy`, `preview`
- [ ] Testar build e dev server

## Dependencias
- Nenhuma (primeira story)

## Notas Tecnicas
- Compatibilidade: `nodejs_compat` flag necessaria no wrangler.toml
- Next.js 15 usa React 19 — verificar compatibilidade com shadcn/ui
- Usar `src/` directory para organizacao

## Arquivos Afetados

- `wrangler.toml`

- `.dev.vars`
- `.gitignore`
