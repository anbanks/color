# Story 11.1 — Verificador de Contraste WCAG

## Descricao
Como designer, preciso verificar se as cores de uma paleta tem contraste suficiente entre si segundo as diretrizes WCAG, para garantir acessibilidade nas minhas escolhas de cores.

## Criterios de Aceitacao
- [ ] Componente de contraste que compara pares de cores da paleta
- [ ] Calcula ratio de contraste WCAG 2.1
- [ ] Indica nivel: AAA (7:1+), AA (4.5:1+), AA Large (3:1+), Fail
- [ ] Visual: badges coloridos (verde AAA, amarelo AA, vermelho Fail)
- [ ] Disponivel na pagina de detalhe da paleta
- [ ] Disponivel no criador de paletas (feedback em tempo real)

## Tarefas
- [ ] Criar `src/components/palette/contrast-checker.tsx`
- [ ] Implementar calculo de contraste em `src/lib/color-utils.ts`
- [ ] Funcao: relativeLuminance(hex) → number
- [ ] Funcao: contrastRatio(hex1, hex2) → number
- [ ] Funcao: wcagLevel(ratio) → 'AAA' | 'AA' | 'AA Large' | 'Fail'
- [ ] Matriz de contraste: todas as combinacoes de pares
- [ ] Integrar na pagina de detalhe da paleta
- [ ] Integrar no criador de paletas

## Dependencias
- Story 3.2 (pagina da paleta)
- Story 4.1 (criador de paletas)

## Notas Tecnicas
- WCAG 2.1 contrast ratio: (L1 + 0.05) / (L2 + 0.05)
- Luminancia relativa: 0.2126*R + 0.7152*G + 0.0722*B (com linearizacao sRGB)
- 4 cores = 6 pares possiveis (combinacoes C(4,2))
- Nao precisa de lib externa — calculo e simples
