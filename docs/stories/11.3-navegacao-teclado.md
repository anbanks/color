# Story 11.3 — Navegacao por Teclado

## Descricao
Como power user, preciso navegar pelas paletas e copiar cores usando apenas o teclado, para ter uma experiencia ultra-rapida sem depender do mouse.

## Criterios de Aceitacao
- [ ] Espaco: gera paleta aleatoria (na home) ou proxima paleta
- [ ] Enter: abre paleta selecionada
- [ ] C: copia todas as cores da paleta selecionada
- [ ] 1-4: copia cor especifica da paleta (posicao 1 a 4)
- [ ] L: toggle like na paleta selecionada
- [ ] S: salvar paleta na colecao
- [ ] Setas cima/baixo: navegar entre paletas no grid
- [ ] Esc: fechar dialogs/voltar
- [ ] Indicador visual de paleta selecionada (borda/highlight)
- [ ] Toast com feedback de cada acao
