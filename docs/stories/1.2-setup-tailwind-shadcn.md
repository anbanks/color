# Story 1.2 — Configurar Tailwind CSS + shadcn/ui + Lucide Icons

## Descricao
Como desenvolvedor, preciso configurar o sistema de UI com Tailwind CSS, componentes shadcn/ui e icones Lucide, para ter uma base de componentes reutilizaveis com design minimalista.

## Criterios de Aceitacao
- [ ] Tailwind CSS v4 configurado e funcionando
- [ ] shadcn/ui inicializado com tema customizado (cores neutras, fundo branco)
- [ ] Fonte Inter configurada via next/font
- [ ] Lucide Icons instalado
- [ ] Componentes base instalados: Button, Card, Dialog, Input, Dropdown, Toast, Tooltip
- [ ] CSS global com reset minimalista
- [ ] Dark mode desabilitado (interface sempre clara)

## Tarefas
- [ ] Verificar se Tailwind ja foi instalado pelo create-next-app
- [ ] `npx shadcn@latest init` — tema: neutral, estilo: default
- [ ] Instalar componentes: `npx shadcn@latest add button card dialog input dropdown-menu toast tooltip`
- [ ] `npm install lucide-react`
- [ ] Configurar fonte Inter em `src/app/layout.tsx`
- [ ] Customizar `globals.css` com cores base (fundo branco, texto cinza escuro)
- [ ] Remover dark mode do tailwind.config

## Dependencias
- Story 1.1 (projeto inicializado)

## Notas Tecnicas
- shadcn/ui sao componentes copy-paste, sem runtime dependency
- Manter tema extremamente limpo — as paletas de cores sao o destaque
- Texto principal: #333333, fundo: #FFFFFF, bordas: #E5E5E5

## Arquivos Afetados
- `components.json`
- `src/components/ui/*`
- `src/app/globals.css`
- `src/app/layout.tsx`
- `tailwind.config.ts`
