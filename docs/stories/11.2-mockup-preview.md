# Story 11.2 — Mini-Preview Mockup de Interface

## Descricao
Como designer, preciso ver um mockup de interface de site que muda de cor em tempo real conforme seleciono uma paleta, para visualizar como as cores ficam na pratica.

## Criterios de Aceitacao
- [ ] Componente mockup de site minimalista (header, botao, card, texto)
- [ ] Cores da paleta aplicadas automaticamente ao mockup
- [ ] Cor 1: fundo/background, Cor 2: header/nav, Cor 3: botoes/accent, Cor 4: texto
- [ ] Atualizacao em tempo real ao mudar cores no criador
- [ ] Disponivel na pagina de detalhe e no criador de paletas
- [ ] Responsivo: mockup redimensiona em mobile

## Tarefas
- [ ] Criar `src/components/preview/site-mockup.tsx`
- [ ] Design do mockup: mini-site com header, hero, 2 cards, botao CTA
- [ ] Mapear cores da paleta para elementos do mockup
- [ ] Transicao suave ao trocar cores (CSS transition)
- [ ] Integrar na pagina de detalhe (`palette/[slug]`)
- [ ] Integrar no criador de paletas

## Dependencias
- Story 3.2 (pagina da paleta)
- Story 4.1 (criador de paletas)

## Notas Tecnicas
- Mockup e puro CSS/HTML — sem iframe ou screenshot
- Mapeamento de cores flexivel: usuario pode ter 3-5 cores
- Fallback se menos de 4 cores: repetir ou usar tons derivados
- Inspiracao: Huemint (preview de interface com paleta)
- Tamanho: ~300x200px, bordas arredondadas, sombra sutil
