# Story 3.1 — Grid de Paletas + Card de Paleta

## Descricao
Como usuario, preciso ver uma grade de paletas de cores na home com cards mostrando 4 faixas de cor empilhadas, para descobrir paletas visualmente de forma rapida.

## Criterios de Aceitacao
- [ ] Componente PaletteCard com 4 faixas de cor verticais
- [ ] Cantos arredondados e sombra sutil no card
- [ ] Hover revela codigos HEX sobre cada faixa
- [ ] Clique na faixa copia o HEX para clipboard com feedback visual (toast)
- [ ] Botao de like (coracao) com contador no card
- [ ] Grid responsivo: 1 col mobile, 2 cols tablet, 3-4 cols desktop
- [ ] Dados carregados do D1 via Server Component
- [ ] Filtro por categoria: Trending, Popular, New, Random

- Story 1.4 (layout base)

## Notas Tecnicas
- Card: w-full, rounded-xl, shadow-sm hover:shadow-md, transition
- Faixa de cor: h-16, first:rounded-t-xl, last:rounded-b-xl
- Hover na faixa: overlay semi-transparente com texto HEX branco
- Like: Server Action para toggle like (POST /api/palettes/[id]/like)
- Trending: ultimas 24h ordenadas por likes
- Popular: all-time ordenadas por likes
- New: ordenadas por createdAt desc
- Random: ORDER BY RANDOM() LIMIT 20
