# Story 10.1 — SEO Avancado (Sitemap, Robots, OG Images)

## Descricao
Como estrategia de SEO, preciso de sitemap dinamico, robots.txt otimizado e OG images para cada paleta, para maximizar a indexacao no Google e o compartilhamento em redes sociais.

## Criterios de Aceitacao
- [ ] `sitemap.ts` gera sitemap XML com todas as paletas publicadas
- [ ] Sitemap inclui URLs nos 3 idiomas com hreflang
- [ ] `robots.ts` configurado (allow all, disallow /admin, /api)
- [ ] OG Image gerada dinamicamente por paleta (4 faixas de cor)
- [ ] OG images armazenadas no Cloudflare R2
- [ ] Metadata completa em cada pagina (title, description, og:image)
- [ ] Sitemap atualizado via ISR (revalidate: 3600)

## Tarefas
- [ ] Criar `src/app/sitemap.ts` com generateSitemaps()
- [ ] Query todas paletas publicadas com slugs
- [ ] Gerar URLs para cada locale (en, pt, es)
- [ ] Criar `src/app/robots.ts`
- [ ] Criar `src/app/palette/[slug]/opengraph-image.tsx` (Next.js OG image generation)
- [ ] Configurar R2 para armazenar OG images geradas
- [ ] Adicionar meta tags og:image na pagina da paleta
- [ ] Testar sitemap com Google Search Console

## Dependencias
- Story 3.2 (paginas de paleta)
- Story 7.1 (i18n com rotas por locale)
- Story 9.1 (paletas publicadas)

## Notas Tecnicas
- Next.js tem suporte nativo a sitemap.ts e robots.ts
- OG Image: usar @vercel/og ou next/og para gerar imagem com as 4 cores
- Sitemap pode ser grande — considerar sitemap index se > 50k URLs
- R2: zero egress fees, ideal para servir OG images
- Testar com: Google Rich Results Test, Facebook Sharing Debugger

## Arquivos Afetados
- `src/app/sitemap.ts`
- `src/app/robots.ts`
- `src/app/palette/[slug]/opengraph-image.tsx`
